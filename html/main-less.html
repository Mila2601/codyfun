В начале на этой странице была первая работа с заголовками, изображениями, видео, аудио.
Трехколоночная разбивка создавалась с использованием свойства CSS display: column. Когда я начала изучение LESS,
я решила полностью переделать эту страницу и сделать здесь слайдшоу на less.
Это было сделано с целью не плодить лишние страницы, поскольку бывшая "главная" стала слишком примитивной.
Файл less написан с использованием Prepros, не откомпилирован в CSS с целью показать исходник.
